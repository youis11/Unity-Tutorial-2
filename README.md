# Version of the game

Sheep Rescue v.1.0

## Description

Shoot hay in order to rescue sheeps and get the highest score possible, if you miss 3 sheeps you lose. This is part of a tutorial, I do not own the visuals of the game.

- ESC (Go to Main Menu).

- A (to go left).
- D (to go right).
